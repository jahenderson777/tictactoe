# tictactoe

Tic Tac Toe technical challenge

## Usage

Run the project directly, via `:main-opts` (`-m hi55.tictactoe`):

    $ clojure -M:run-m

Run the project's tests (they'll fail until you edit them):

    $ clojure -M:test:runner

Build an uberjar:

    $ clojure -X:uberjar


Run that uberjar:

    $ java -jar tictactoe.jar
